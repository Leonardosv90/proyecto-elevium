# Decisiones del proyecto

## Base inicial

Fecha: 2026-05-29

Se toma como base de desarrollo la version limpia generada desde `outputs/elevium-publicable`.

## Criterios actuales

- Mantener el sitio como estatico por ahora.
- Priorizar conversion hacia diagnostico estrategico.
- Conservar el posicionamiento de autoridad digital con estrategia, contenido, diseno, IA y automatizacion.
- No volver a incluir assets no usados dentro del proyecto vivo.
- Conectar formularios mediante `data-endpoint` cuando se defina la herramienta: Google Sheets, Formspree, Make, Zapier, CRM o backend propio.

## Pendientes de decision

- Plataforma de publicacion.
- Destino real de formularios.
- Politica de privacidad y terminos.
- Casos reales o demostrativos.
- Si el sitio se mantiene estatico o migra a Astro/Next en una fase posterior.
